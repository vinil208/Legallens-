# LegalLens - AI Legal Assistant for India

An AI-powered legal assistance platform designed to help Indian citizens understand their rights, generate legal documents, and find relevant resources in multiple Indian languages.

## Features

- **AI Legal Advisor**: Get instant guidance on legal issues in your preferred language
- **Document Generator**: Create RTI applications, legal notices, and complaint letters
- **Resource Finder**: Locate free legal aid offices, government helplines, and local authorities
- **Multi-language Support**: Available in English, Hindi, Tamil, Telugu, Kannada, and Malayalam
- **Voice Input**: Speak your legal issues for easier access

## Technologies Used

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **Backend**: Supabase for data management
- **AI Integration**: Google Gemini AI for legal guidance

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <YOUR_GIT_URL>
cd legal-access-india
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```bash
npm run build
```
